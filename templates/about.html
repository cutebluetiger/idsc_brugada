About This Tool

An expert-facing clinical decision-support system for Brugada syndrome detection using machine learning and explainable AI.

Purpose

Analyse 12-lead ECG recordings and provide an interpretable Brugada vs Normal prediction. The tool augments — never replaces — expert clinical judgment.

Target Users

Cardiologists, electrophysiologists, clinical researchers and technical evaluators who can interpret results in proper clinical context.

Input Format

WFDB format ECG recordings (.hea + .dat file pairs) containing 12-lead digital signals sampled at standard clinical rates.

Preprocessing

60 Hz notch filter for powerline noise, high-pass filter at 0.5 Hz for baseline wander, low-pass filter at 45 Hz for high-frequency artefacts.

Feature Engineering

Time-domain (mean, RMS, kurtosis…), frequency-domain (spectral power, entropy), and nonlinear (Hjorth parameters) features per lead, plus cross-lead correlations.

ML Model

Primary classifier is XGBoost, chosen for strong tabular performance and native SHAP compatibility. Random Forest serves as a benchmark.

Explainability

SHAP (SHapley Additive exPlanations) values decompose each prediction into per-feature contributions, enabling transparent reasoning.

Safety Notice

This is a research / decision-support tool only. It is not a certified medical device. Predictions must be validated by qualified clinicians before any clinical action.
